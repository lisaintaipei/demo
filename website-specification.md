## 技術實現

### 框架與工具
- React (TypeScript)
- Inline styles (完全符合規格的顏色和尺寸)
- Unsplash API (圖片來源)

### 組件結構
```
App.tsx
├── Header.tsx
├── InfoSection.tsx
├── ServiceSection.tsx
│   └── ServiceCard.tsx
├── ContactSection.tsx
└── Footer.tsx
```

### 互動功能
- 平滑滾動 (smooth scrolling)
- Sticky header (固定導覽列)
- 表單狀態管理 (React useState)
- 圖片載入錯誤處理 (ImageWithFallback 組件)
